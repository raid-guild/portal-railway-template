# Community Portal Template Docs

This template is a deployable Payload + Next.js portal for communities, programs, and contributor networks.

The default product model is intentionally small:

- `Brief`: current snapshot of what is happening.
- `Project`: a visible collaboration surface.
- `Thread`: an ongoing line of work or thought.
- `Activity Item`: a dated factual signal.
- `Event`: a scheduled session or calendar anchor.
- `Profile`: a person or contributor identity.

Start with these primitives before adding new collections. New feature areas should become modules only when they need their own lifecycle, permissions, records, or API surface.
