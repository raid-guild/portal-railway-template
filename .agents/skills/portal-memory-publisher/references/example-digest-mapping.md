# Example Digest Mapping

Input: Weekly community planning meeting digest.

Useful output plan:

1. Update an existing project when the meeting clearly advanced a collaboration surface.
2. Create one or more activity items for dated facts, decisions, blockers, or contributions.
3. Link the activity to related profiles, threads, projects, or events when source data supports it.
4. Draft a daily brief section only when there are enough sourced signals to summarize.

Example records:

- `events`: `Community Planning Session`
- `projects`: `Community Portal Starter`
- `threads`: `Onboarding Improvements`
- `activityItems`: `Planning session clarified the next onboarding form iteration`

Keep names, people, dates, and URLs exactly grounded in the source material.
